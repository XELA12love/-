# -
本文设计并实现了一款轻量化电子产品推荐系统，针对传统电商检索模式存在的信息过载问题，为小型数码销售平台提供商品管理与个性化推荐一体化解决方案。系统采用主流前后端分离技术栈，后端基于 Spring Boot、MyBatis Plus、MySQL 开发，依靠 JWT 实现安全登录与接口权限拦截；前端使用 Vue.js、Element UI 和 Axios 搭建交互界面。平台支持用户注册登录、商品浏览收藏、下单交易，同时为管理员提供商品、分类、用户等后台管理功能。系统融合两种协同过滤算法实现智能推荐，依据用户消费行为量化兴趣偏好，通过算法融合、类目优化提升推荐多样性，并设置分层兜底策略应对新用户、新商品冷启动难题。系统界面简洁、响应高效，适配中小规模数码电商使用场景，具备较强的实用性与参考价值。
